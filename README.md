## Description

Backend du projet chef d'oeuvre TRINITY APP

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ nest start

# watch mode
$ nest start --watch

# build 
nest build

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Variables d'environnement

```bash
# Database connection
$ DB_CONNECTION

# Port
$ PORT

```
